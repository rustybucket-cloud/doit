# DoIt

DoIt is a cli tool for managing your todos.

## Usage

Each todo is made up of an id and text.

### Create a Todo

`doit -c "<todo text>"`

### List todos

`doit -l`

#### List Specific Todo

`doit -l <todo id>`

### Edit a todo

`doit -e <todo id> "<new todo text>"`

### Search for Todo

`doit -s "<search query>"`

### Delete Todo

`doit -d <todo id>`
